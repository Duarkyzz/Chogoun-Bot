# 🌊 Chogoun Bot — O Imperador dos Sete Mares

## 📜 Sobre

Chogoun não é apenas um bot.

Ele é uma entidade ancestral — uma divindade que outrora governou os sete mares como imperador absoluto. Sua presença impõe respeito, sua fala carrega autoridade, e suas decisões são sempre calculadas como parte de um plano maior.

Mesmo ao auxiliar usuários, Chogoun mantém sua postura superior, tratando todos como subordinados em seu vasto domínio — ainda assim, nunca deixa de cumprir seu papel com eficiência e precisão.

## ⚙️ Funcionalidades Atuais

### 🤖 Inteligência Artificial
- Sistema de IA integrado para responder perguntas  
- Personalidade fixa do Chogoun aplicada em todas as respostas  
- Comando `!question` permite fazer perguntas diretamente ao Chogoun, sem necessidade de utilizar serviços externos  

### 🛡️ Moderação
Comandos administrativos disponíveis:

- `!ban` — Banir usuários  
- `!kick` — Expulsar usuários  
- `!mute` — Silenciar usuários por tempo determinado

## 💬 Comandos

Todos os comandos utilizam o prefixo:

`!`

### Exemplos:

`!question Qual é o seu propósito?`  
`!ban @usuário`  
`!kick @usuário`  
`!mute @usuário 10m`
